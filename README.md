kuler2gpl
===========

## What is it?

Converts Adobe Kuler ASE Color swatches to the GIMP / Inkscape GPL format.


## Why?

There wasn't a tool available for Linux, so here's one.


## Install

	$ npm install -g kuler2gpl


## Usage

	$ kuler2gpl -i /path/to/input/directory


## Help

	$ kuler2gpl --help
